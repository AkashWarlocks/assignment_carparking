# Vehicle Parking Slot Allotment System 

# Technology stack:
  1. Backend - Nodejs
  2.Frontend - Angular
  3.Database - MongoDB
  4.Server - Heroku
  
# Working - 
  1. # Slot allotment - 
      - Used Nodejs and Advance Mongo Queries(aggregations) for finding empty slots on following conditions
        1. Level should be filled linearly
        2. Motorcycle - 
          1.1 Slot type - ANY
          1.2 Space - 1 slot
        3. Car - 
          1.1 Slot Type - Large, Compact
          1.2 Space - 1 slot
        4. Bus - 
          1.1 Slot Type - Large
          1.2 Space - 5 Slots
